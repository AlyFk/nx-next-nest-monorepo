

# NX Monorepo Project

## About The Project

This is a monorepo build with Nx, in apps/ you will find a Next.js client and a Nest.js server

### Built With

* [Nx](https://nx.dev).
* [TypeScript](https://www.typescriptlang.org/)
* [Next.js](https://nextjs.org/)
* [React.js](https://reactjs.org/)
* [Nest.js](https://nestjs.com/)

## Installation

```
npm install
```

## Development

```
npm start
```

API
- `npm run dev:api`

Client
- `npm run dev:client`

## Production

```
npm build
```

API
- `npm run build:api`

Client
- `npm run build:client`

## Running Tests

```
npm test
```

API
- `npm run test:api`

Client
- `npm run test:client`
